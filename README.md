## Grocery Shopping API

### Setup

1. Install dependencies:
    ```bash
    docker-compose up -d
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

4. Build the project:
    ```bash
    npm run build 
    ```

4. Start the server:
    ```bash
    npm run start
    ```

### Usage

- Access the API endpoints to manage grocery shopping lists and items.
